Installing the Tackle Operator

Download and install the Tackle Operator from operatorhub.io.

Prerequisites

  • You must have admin privileges to install the Tackle Operator.

Procedure

  1. Navigate to operatorhub.io.

  2. Click the Modernization & Migration category.

  3. Click the Tackle Operator tile and then click Install.

  4. Follow the installation instructions.

tackle-documentation is maintained by konveyor.
