Installing the Tackle application

Install the Tackle application in a project.

Prerequisites

  • The Tackle Operator must be installed on the cluster.

  • You must have project-admin privileges to install the Tackle application in a project.

  • You must create a project for Tackle.

Procedure

  1. Install the Tackle application in a project by following the procedure for your enterprise Kubernetes platform.

  2. Obtain the route or URL of the tackle-ui pod to open the Tackle web console in a new browser window.

tackle-documentation is maintained by konveyor.
