Logging in to the Tackle web console

You can log in to the Tackle web console.

Prerequisites

  • The Tackle application must be installed.

  • You must have the route of the tackle-ui pod. This is the Tackle web console URL.

Procedure

  1. In a browser, navigate to the Tackle web console.

  2. Enter tackle in the Username or email field, password in the Password field, and then click Log in to access the Tackle web console.

    You can review the services of the Tackle web console.

Important

You must change the default password of the tackle user.
